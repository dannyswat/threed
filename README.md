# Dummy House Walker

A small Godot 4 3D prototype where a simple rigged dummy character walks around a house made from procedural placeholder geometry.

## What is included

- A controllable third-person dummy character
- A simple bone-based rig built in code for the character body
- A furnished house built from box meshes and collision shapes
- A trailing third-person camera
- No external art assets or add-ons

## Controls

- `WASD` or arrow keys: move

## Run

1. Open the folder in Godot 4.
2. Let Godot import the project.
3. Run the main scene.

The project entry point is `res://scenes/Main.tscn`.
